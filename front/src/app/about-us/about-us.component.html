Le projet consiste à créer un site web interactif permettant aux utilisateurs de participer à une compétition du célèbre jeu Pong.
Ce site web offre une expérience de jeu multijoueur en temps réel, accompagnée d'une interface utilisateur intuitive et d'un chat intégré.
Pour garantir la sécurité des utilisateurs, le backend est développé en utilisant le framework NestJS, tandis que le frontend est réalisé avec un framework TypeScript de choix.
Une base de données PostgreSQL est utilisée pour stocker les informations nécessaires.
Le site web offre une gamme de fonctionnalités pour améliorer l'expérience de jeu.

Fonctionnalités du site :

  • Connexion via le système OAuth de l'intranet 42
  • Personnalisation du profil avec un nom d'utilisateur unique et un avatar
  • Activation de l'authentification à deux facteurs (2FA)
  • Système de chat avec channels publics, privés et protégés par mot de passe
  • Envoi de messages directs
  • Ajout d'amis et blocage d'autres utilisateurs
  • Consultation des profils des joueurs
  • Jeu Pong multijoueur en temps réel
  • Système de matching automatique pour trouver des adversaires
  • Options de personnalisation du jeu

Le projet met également l'accent sur la sécurité. Les mots de passe sont stockés de manière sécurisée grâce au chiffrement, et des mesures sont prises pour prévenir les injections SQL. Une validation côté serveur est mise en place pour les formulaires et les requêtes utilisateur.

En résumé, ce projet vise à offrir une expérience de jeu interactive et compétitive du célèbre jeu Pong. Avec des fonctionnalités telles qu'un système de chat, des comptes utilisateur personnalisables et un jeu Pong multijoueur en temps réel, les utilisateurs pourront profiter d'une expérience de jeu immersive et divertissante.
